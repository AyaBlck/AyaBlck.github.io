Overview

Disinformation has become a defining challenge of the digital age. This project compares two fundamentally different AI approaches to fake news detection: BERT (a discriminative model, fine-tuned on labelled data) and LLaMA 3 (a generative model, using few-shot prompting via Ollama). The goal was to determine which approach is more suitable for deployment in a news organisation’s editorial pipeline.

Dataset

The ISOT Fake and Real News Dataset contains 44,919 labelled news articles collected from Reuters (real) and Politifact-flagged sources (fake).

LLaMA 3 Few-Shot Approach

LLaMA 3 was run locally using Ollama — an open-source runtime for large language models. No model weights were updated; instead, a few-shot prompt with three labelled examples was used to guide classification.

Code

Key Findings

TipMain Insights
  • BERT achieved 99.75% accuracy with perfect recall on fake articles — only 2 out of 800 articles were misclassified
  • LLaMA 3 achieved just 54% accuracy (barely above random chance) with a strong bias towards predicting “Real” — 23 out of 25 fake articles were missed
  • The performance gap is not a failure of LLaMA 3 as a model, but a limitation of few-shot prompting versus gradient-based fine-tuning for structured classification tasks
WarningBusiness Recommendation

For deployment in a news organisation’s editorial pipeline, BERT is the clear choice — its 99.75% accuracy translates to roughly 25 misclassifications per 10,000 articles. However, a hybrid architecture combining BERT’s classification reliability with LLaMA 3’s generative explainability could offer the best of both worlds: BERT flags articles, LLaMA 3 explains why.

Ethical Considerations

This project evaluated both models against the EU AI Act (2024) and Trustworthy AI principles. Key considerations include maintaining human oversight in editorial decisions, addressing dataset bias (the ISOT dataset reflects Western media perspectives), and ensuring transparency in model decision-making. LLaMA 3’s local deployment via Ollama offers a privacy advantage, as no article text leaves the organisation’s infrastructure.
